# Models

This folder is for storing pretrained or fine-tuned transformer models used for:
- Natural language to SQL generation (e.g., T5, BART)
- Intent classification (e.g., BERT)

## Recommended Datasets
- [Spider](https://yale-lily.github.io/spider)
- [WikiSQL](https://huggingface.co/datasets/wikisql)

## Usage
Download or fine-tune models and place them here as needed.
